# Web Portfolio
https://fyecobain.github.io

> React + TypeScript + Vite

## Get this repo
```console
git clone https://github.com/FyeCobain/FyeCobain.github.io.git
```
```console
cd FyeCobain.github.io
```
```console
pnpm install
```
#### Run the build preview
```console
pnpm run prev
```
#### Run the development server
```console
pnpm run dev
```
